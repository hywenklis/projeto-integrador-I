Memória Cache

A memória cache é uma pequena quantidade de memória rápida que armazena temporariamente dados frequentemente acessados ​​pela CPU para reduzir o tempo de acesso à memória principal, melhorando assim o desempenho do sistema. A memória cache opera por meio de um conjunto de algoritmos que determinam qual informação é armazenada em cache e por quanto tempo, bem como a maneira como os dados são organizados na cache para maximizar a eficiência do acesso à memória. A existência da memória cache é transparente para o software, pois o hardware gerencia automaticamente o processo de armazenamento e recuperação dos dados em cache. Existem diferentes níveis de cache em um sistema, desde a cache L1 embutida diretamente na CPU até a cache L3 compartilhada por vários núcleos da CPU. O tamanho, a organização e o desempenho da memória cache são fatores importantes no design e desempenho de processadores e sistemas de computação em geral.
